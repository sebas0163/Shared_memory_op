# OS - Project 1

### How to install dependencies (in a Linux environment):

```
sudo apt-get install libgtk-3-dev
```

### How to compile project:

```
make
```

### How to run creator process:

```
make run_creator
```

### How to run client process:

```
make run_client
```

### How to run recreator process:

```
make run_recreator
```
